SE COMPILA UNA VEZ Y SE EJECUTA UNA VEZ POR CADA PARTIDA

JUEGUE DE ACUERDO A LA LISTA DE PAREJAS

UNA VEZ, DESPUES SE VULEVE A EJECUTAR PARA LAS OTRAS 
PARTIDAS UNA VEZ.
Este es el primer juego por lo que 
para cada partida como en las instrucciones
se especifica se ponen las Claves de los Jugadores
** java MiniBlack Jugador1 Jugador2
Cada partida solo es neceseario poner de nuevo
los Jugadores para la ejecuacion de la misma
** java MiniBlack Jugador3
Se ejecuta de nuevo cada que se juega una partida
